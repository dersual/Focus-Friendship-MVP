# Focus Friends MVP

This project is the Minimum Viable Product (MVP) for "Focus Friends," a web application that gamifies focus sessions using a digital pet.

## Project Goal

The goal is to build a functional and visually appealing prototype that includes the core features of the application.

## Technology Stack
(Information below is for developers)

-   **Frontend:** React (with JavaScript)
-   **Styling:** Bootstrap and custom CSS, following Material Design principles.
-   **Backend:** Node.js with Express.js
-   **Package Manager:** npm

## Key Features

-   Pomodoro Timer
-   "Cutie Bean" digital companion that evolves
-   XP and Leveling System
-   Customization Shop
-   Friends List

## Getting Started

Follow these instructions to set up and run the project locally.

### Prerequisites

-   Node.js (LTS version recommended)
-   npm (comes with Node.js)

### Installation

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd Focus-Friends-MVP-1
    ```

2.  **Frontend Setup:**
    Navigate to the `client` directory and install dependencies:
    ```bash
    cd client
    npm install
    ```

3.  **Backend Setup (Coming Soon):**
    Navigate to the `server` directory and install dependencies:
    ```bash
    cd ../server
    npm install
    ```
    *(Note: The backend setup is not yet implemented in this MVP phase.)*

### Running the Application

1.  **Start the Frontend:**
    From the `client` directory:
    ```bash
    npm start
    ```
    This will open the React application in your browser (usually at `http://localhost:3000`).

2.  **Start the Backend (Coming Soon):**
    From the `server` directory:
    ```bash
    npm start
    ```
    *(Note: The backend server is not yet implemented in this MVP phase.)*
